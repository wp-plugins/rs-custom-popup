Allows you to create a popup for your website with specific input to taylor for your needs

== Description ==

Create a popup for your website. You are able to specify the following in order to tailor to your needs.

*	Image (This will be the image that shows as the popup and is) `required`
*	Link (This will the link the user is taken to upon clicking your popup)
*	Pages (This is how you specify what page(s) you require your popup to be shown on) `required`
*	Cookie (Specify whether you would like to drop a cookie on the users machine if the close the popup in order for it to stay hidden)

== Installation ==

1. Upload `rs-custom-popup` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
